# Backend Claude Guide

Read `AGENTS.md` first.

This repo contains the business logic for the finance tracker. If you are unsure where to start, default to:

1. `docs/backend-architecture.md`
2. `docs/backend-api-map.md`
3. The relevant `routes/`, `controllers/`, and `models/` files for the feature

Mandatory rule: if you change routes, models, auth behavior, env requirements, or domain rules, update the docs in the same change.
